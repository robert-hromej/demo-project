# RecipeMatch - Огляд проєкту

## Вступ

**RecipeMatch** - це розумний застосунок для пошуку рецептів, розроблений для українських користувачів. Застосунок допомагає людям знаходити рецепти на основі наявних інгредієнтів або бюджетних обмежень, роблячи планування їжі простішим та економнішим.

## Основна концепція

Застосунок відповідає на два фундаментальні питання:

1. **"Що я можу приготувати з того, що маю?"** (пошук за інгредієнтами)
   > "В мене є молоко, 2 яйця, пів хліба - покажи що можна приготувати"

2. **"Що я можу приготувати з моїм бюджетом?"** (пошук за бюджетом)
   > "В мене 200 грн - що приготувати?"

## Ключові можливості

### Можливості MVP

#### 1. Пошук рецептів за інгредієнтами
- Введення наявних інгредієнтів
- Пошук рецептів, які можна приготувати з цих інгредієнтів
- Показ кількості відсутніх інгредієнтів
- Фільтрація за відсотком збігу (наприклад, "є 80% інгредієнтів")

#### 2. Пошук рецептів за бюджетом
- Встановлення бюджету в UAH
- Пошук рецептів у межах бюджету
- Показ орієнтовної вартості порції
- Сортування за економічністю

#### 3. Управління рецептами
- Перегляд рецептів за категоріями
- Перегляд детальної інформації про рецепт:
  - Інгредієнти з кількістю
  - Покрокові інструкції
  - Час приготування
  - Розмір порції
  - Орієнтовна вартість

#### 4. Оцінки та відгуки користувачів
- Оцінювання рецептів (1-5 зірок)
- Залишення текстових відгуків
- Сортування за популярністю/рейтингом
- Розрахунок середньої оцінки

#### 5. Навігація за категоріями
- Ієрархічні категорії (наприклад, Сніданки > Каші > Вівсянка)
- Фільтрація рецептів за категорією
- Рекомендовані/сезонні категорії

### Майбутні можливості (Фаза 2+)

#### Інтеграція з супермаркетами
- **Інтеграція з Silpo**: Ціни в реальному часі, наявність
- **Інтеграція з ATB**: Бюджетні варіанти
- Автоматичне оновлення цін
- Генерація списку покупок з посиланнями на магазини

#### Розширені можливості
- Персоналізовані рекомендації
- Дієтичні обмеження (вегетаріанське, безглютенове тощо)
- Планування меню на тиждень
- Інформація про харчову цінність
- Рецепти від користувачів
- Соціальний обмін

## Цільова аудиторія

1. **Економні кухарі** - Студенти, молоді сім'ї
2. **Зайняті професіонали** - Швидке планування їжі
3. **Домашні кухарі** - Вирішення проблеми "що в холодильнику"
4. **Україномовні користувачі** - Підтримка рідної мови

## Бізнес-цінність

- **Для користувачів**: Економія грошей, зменшення харчових відходів, відкриття нових рецептів
- **Для майбутнього бізнесу**: Партнерські програми з супермаркетами, преміум-функції

## Технічні особливості

- Сучасний Rails 8.1.2 API backend
- React 19 frontend з TypeScript
- Пошук у реальному часі з розумними алгоритмами збігу
- Масштабована архітектура для майбутніх інтеграцій
- Робота з грошима в українській гривні (UAH)
